# Домашнее задание к семинарам 10-11 (HW10-11)

Тема: компьютерное зрение в PyTorch. Часть S10: CNN, аугментации, transfer learning на ResNet. Часть S11: детекция или сегментация, корректная визуализация и базовая валидация.

---

## 1. Кратко: что сделано

- **Часть A (классификация)**: Выбран датасет **STL10** — содержит 10 классов объектов, изображения 96x96, подходит для демонстрации CNN и transfer learning. Датасет имеет отдельный test split, что позволяет корректно оценить финальное качество.
- **Часть B (structured vision)**: Выбран датасет **OxfordIIITPet**. Выбран трек **segmentation**. Этот датасет предоставляет качественные маски для сегментации домашних животных, что позволяет наглядно продемонстрировать работу pretrained модели и постобработку масок.
- **Сравнения**: В части A сравнивались простая CNN с аугментациями и без, а также ResNet18 в режимах head-only и fine-tune. В части B сравнивались два режима постобработки масок сегментации.

## 3. Данные

### 3.1. Часть A: классификация

- **Датасет**: `STL10`
- **Разделение**: train/val/test (80/20 для train/val, отдельный test от torchvision)
- **Базовые transforms**: Resize(96x96), ToTensor, Normalize
- **Augmentation transforms**: RandomHorizontalFlip, RandomRotation(10), ColorJitter
- **Комментарий**: STL10 содержит 10 классов (самолёт, автомобиль, птица и т.д.), 5000 train и 8000 test изображений. Изображения цветные 96x96, задача средней сложности для MLP, но хорошо подходит для CNN.

### 3.2. Часть B: structured vision

- **Датасет**: `OxfordIIITPet`
- **Трек**: `segmentation`
- **Что считается ground truth**: Пиксельные маски с классами объектов (животное, фон, границы)
- **Какие предсказания использовались**: Выход DeepLabV3-ResNet50 с pretrained весами из COCO
- **Комментарий**: Датасет содержит 37 классов пород кошек и собак с качественными масками сегментации. Для упрощения задачи использована бинарная сегментация (foreground = животное, background = всё остальное).

---

## 4. Часть A: модели и обучение (C1-C4)

- **C1 (simple-cnn-base)**: SimpleCNN (3 conv блока + FC) без аугментаций, базовые трансформы
- **C2 (simple-cnn-aug)**: Та же архитектура CNN, но с аугментациями (flip, rotation, color jitter)
- **C3 (resnet18-head-only)**: ResNet18 с pretrained ImageNet весами, backbone заморожен, обучается только fc-слой
- **C4 (resnet18-finetune)**: ResNet18 с pretrained весами, разморожены layer4 + fc для fine-tuning

**Дополнительно**:

- **Loss**: CrossEntropyLoss
- **Optimizer(ы)**: Adam (lr=1e-3 для CNN, lr=1e-3 для C3, lr=1e-4 для C4)
- **Batch size**: 64
- **Epochs (макс)**: 15
- **Критерий выбора лучшей модели**: best_val_accuracy

---

## 5. Часть B: постановка задачи и режимы оценки (V1-V2)

### Выбран segmentation track

- **Модель**: DeepLabV3-ResNet50 с pretrained весами (COCO_WITH_VOC_LABELS_V1)
- **Что считается foreground**: Пиксели, принадлежащие классу животного (классы 1-36), фон = класс 0
- **V1**: Базовая постобработка — argmax по выходу модели для получения маски
- **V2**: Альтернативная постобработка — пороговая бинаризация для foreground-класса с threshold=0.5
- **Как считался mean IoU**: Для каждого класса (foreground/background) вычисляется IoU = intersection/union, затем усредняется
- **Считались ли дополнительные pixel-level метрики**: Нет, только mean IoU для простоты

## 7. Анализ

Простая CNN (C1/C2) показывает ограниченную точность на STL10 из-за малого размера архитектуры и отсутствия pretrained знаний. Аугментации дают устойчивое улучшение, особенно снижая переобучение на train данных.

Pretrained ResNet18 значительно превосходит простую CNN благодаря признакам, обученным на ImageNet. Head-only подход (C3) работает хорошо, но fine-tuning (C4) позволяет адаптировать верхние слои к специфике STL10, что даёт дополнительное улучшение.

Для сегментации выбрана метрика Mean IoU, так как она корректно оценивает качество пересечения масок и широко используется в benchmark'ах. Переход от V1 к V2 показал, что стандартная argmax-постобработка работает лучше пороговой для данной модели и датасета.

Наиболее показательные ошибки: путаница между похожими классами в классификации (например, кошки разных пород) и неточные границы в сегментации на сложных фонах.

## 8. Итоговый вывод

- **Базовый конфиг классификации**: ResNet18 с partial fine-tuning (C4) — лучшее соотношение качества и сложности обучения.
- **Главное про transfer learning**: Pretrained веса дают значительное преимущество даже при заморозке большинства слоёв, а fine-tuning верхних слоёв позволяет адаптировать модель к целевому датасету.
- **Главное про segmentation**: Корректный выбор метрики (IoU для сегментации) и постобработки критически важен для интерпретации результатов. Pretrained модели из torchvision позволяют быстро получить работающий пайплайн без дообучения.
